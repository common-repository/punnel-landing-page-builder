Landing Page Builder WordPress Plugin. 
Connector to access content from Punnel service. 
Powerful Landing Page Builder for marketing events
<a href="https://Punnel.com" target="_blank">https://Punnel.com</a>

== Installation ==

1. Go to Plugins in your WordPress administration, click Add New, and search for “Punnel WordPress Plugin”.
2. Now click at Install Now button.
3. Click on Activate after upload has completed.
4. Go to Settings and choose Punnel.
5. Log into your Punnel account.
6. When publishing your landing page from Punnel, choose WordPress
